Why You're Seeing This Page

üîß Configuration Needed

This project uses TypeScript with CommonJS modules, which don't run directly in browsers.

To deploy the P2P game, you need to:

Option 1: Use a Bundler (Recommended)

1. Install a bundler (Vite or Webpack):

npm install --save-dev vite


2. Update tsconfig.json to use ES modules:

"module": "ES2020" instead of "commonjs"


3. Create vite.config.js:



4. Add to package.json scripts:

"dev": "vite"

"build": "vite build"


5. Build and deploy:

npm run build

git add dist

git commit -m "Build for GitHub Pages"

git push


6. Configure GitHub Pages:

Go to repo Settings ‚Üí Pages ‚Üí Source: Deploy from branch ‚Üí Branch: main ‚Üí Folder: /dist

Option 2: Use CDN (Quick but Limited)

Load PeerJS directly via CDN and write inline JavaScript (no build step).

See the example in src/p2p/BROWSER_ROOM.md

Option 3: GitHub Actions Auto-Deploy

Create .github/workflows/deploy.yml to automatically build and deploy on push.
